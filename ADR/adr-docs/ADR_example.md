
# 001-framework : choix du framework frontend

## Contexte
Nous devons choisir un framework pour développer la plateforme.

## Décision
Nous retenons **React** pour sa communauté, ses outils et sa compatibilité avec notre stack.

## Alternatives
- Angular : trop complexe pour notre besoin
- Vue.js : intéressant mais moins répandu dans l’équipe

## Conséquences
- Positif : adoption rapide par l’équipe
- Négatif : nécessité de gérer la migration de certains composants

Validé
